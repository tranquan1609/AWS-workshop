# Summary Report: “First Cloud AI Journey – DevOps & Career Development Event”

### Event Objectives

- Share the real role and daily work of a DevOps Engineer in enterprise environments.
- Introduce multinational corporate culture and skills needed for career growth.
- Guide the design of a scalable URL Shortener system on AWS.
- Inspire learning and career development from First Cloud AI Journey to AWS Partner.

### Speakers

- **Mr. Trong H. Truong** (DevOps Engineer @ Endava Vietnam) — *What does a DevOps Engineer really do?*
- **Mr. Dat Pham** (Data Analytics Engineer) & **Mr. Cuong Nguyen** (Manufacturing Excellence Engineer) — *Real stories and culture at multinational corporations*
- **Mr. Dinh Trung Kien** & **Mr. Nguyen Minh Tho** — *A scalable URL shortening service on AWS*
- **Mr. Danh Hoang Hieu Nghi** (AI Engineer — AWS Community Builder — AWS Student Builder Group Leader) — *From First Cloud AI Journey to AWS Partner*

### Key Highlights

#### What does a DevOps Engineer really do?

- Introduced the DevOps Engineer role in software development and operations.
- Shared daily work: CI/CD, infrastructure management, system monitoring, and deployment automation.
- Introduced common tools: Git, Docker, Kubernetes, Jenkins, and AWS services.
- Shared learning paths and skills needed for a DevOps career.

#### Real stories and culture at multinational corporations

- Shared experience working in international enterprise environments.
- Introduced professional culture, collaboration, and accountability in organizations.
- Analyzed the importance of communication, time management, and teamwork.
- Advice for students preparing to enter the corporate world.

#### A scalable URL shortening service on AWS

- Introduced URL shortener architecture on AWS.
- Analyzed scalable, highly available system design.
- Applied AWS services for request handling, data storage, and traffic management.
- Shared solutions for performance, cost, and security optimization.

#### From First Cloud AI Journey to AWS Partner

- Shared the learning journey from First Cloud AI Journey to AWS Partner.
- Introduced learning opportunities, AWS certifications, and student support programs.
- Shared experience building Cloud learning paths and skill development.
- Motivated students to learn proactively and join tech communities.

**Week 2:** 27 April – 1 May 2026

### Week 2 Objectives

- Learn foundational AWS services for building and deploying systems.
- Understand how to set up network infrastructure with Amazon VPC and deploy servers on Amazon EC2.
- Practice database management with Amazon RDS.
- Get familiar with auto scaling using Auto Scaling Group and Elastic Load Balancer.
- Learn how to manage and control AWS service costs.

### Week 2 Outcomes

- Understood Amazon VPC's role in building private network infrastructure on AWS, including VPC, Subnet, Route Table, Internet Gateway, and Security Group.
- Gained foundational Amazon EC2 knowledge: launching instances, choosing configurations, and deploying applications on Cloud.
- Successfully practiced Amazon RDS: creating and managing relational databases without directly administering server infrastructure.
- Understood Amazon EC2 Auto Scaling Group and Elastic Load Balancer for auto scaling and traffic distribution to improve performance and availability.
- Learned to use AWS Budgets for cost tracking, budget setup, and alerts for better cost control and optimization.
- Reinforced knowledge of combining Networking, Compute, Database, Auto Scaling, and Cost Management services when building and operating systems on AWS.
- Improved AWS deployment and management skills through hands-on labs, with a clearer process for building scalable, secure, and cost-optimized Cloud systems.

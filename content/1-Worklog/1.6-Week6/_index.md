**Week 6:** 25 May – 29 May 2026

### Week 6 Objectives

- Learn enterprise storage solutions on AWS.
- Practice file sharing with AWS Storage Gateway and Amazon FSx.
- Strengthen web application security knowledge with AWS Web Application Firewall (WAF).
- Learn to manage AWS resources with Tags and Resource Groups.
- Apply AWS IAM for access control based on Resource Tags.

### Week 6 Outcomes

- Understood AWS File Storage Gateway for connecting on-premises storage to AWS Cloud with data sync and file sharing.
- Successfully practiced Amazon FSx for Windows File Server for Windows file storage and access management.
- Learned AWS WAF for building security rules against SQL Injection and XSS attacks.
- Used AWS Tags and Resource Groups to organize, manage, and search resources for cost tracking and administration.
- Applied AWS IAM with Resource Tags for detailed access policies on EC2 instances with Least Privilege.
- Reinforced Storage, Security, Resource Management, and IAM knowledge for secure, well-organized enterprise AWS infrastructure.

**Tuần 6:** 25/05/2026 – 29/05/2026

### Mục tiêu tuần 6

- Tìm hiểu các giải pháp lưu trữ doanh nghiệp trên AWS.
- Thực hành triển khai hệ thống chia sẻ tệp với AWS Storage Gateway và Amazon FSx.
- Nâng cao kiến thức về bảo mật ứng dụng Web bằng AWS Web Application Firewall (WAF).
- Học cách quản lý tài nguyên AWS bằng Tags và Resource Groups.
- Áp dụng AWS IAM để kiểm soát quyền truy cập dựa trên Resource Tags.

### Kết quả đạt được tuần 6

- Hiểu được vai trò của AWS File Storage Gateway trong việc kết nối hệ thống lưu trữ tại chỗ với AWS Cloud, đồng thời nắm được cơ chế đồng bộ dữ liệu và chia sẻ tệp giữa môi trường On-premises và Cloud.
- Thực hành thành công Amazon FSx for Windows File Server, hiểu cách triển khai hệ thống lưu trữ tập tin dành cho môi trường Windows và quản lý quyền truy cập vào dữ liệu.
- Nắm được chức năng của AWS Web Application Firewall (WAF), biết cách xây dựng các quy tắc bảo mật nhằm bảo vệ ứng dụng Web trước các hình thức tấn công phổ biến như SQL Injection và Cross-Site Scripting (XSS).
- Biết cách sử dụng AWS Tags và Resource Groups để tổ chức, quản lý và tìm kiếm tài nguyên AWS hiệu quả, đồng thời hỗ trợ việc theo dõi chi phí và quản trị hệ thống.
- Thực hành áp dụng AWS IAM kết hợp với Resource Tags để xây dựng chính sách phân quyền chi tiết, giúp kiểm soát quyền truy cập vào các EC2 Instance theo nguyên tắc Least Privilege.
- Củng cố kiến thức về Storage, Security, Resource Management và Identity & Access Management, đồng thời nâng cao khả năng xây dựng và quản trị hạ tầng AWS theo hướng an toàn, khoa học và đáp ứng các yêu cầu của doanh nghiệp.

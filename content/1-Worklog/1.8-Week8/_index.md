### Week 8 Outcomes

- Continued strengthening Data Lake knowledge on AWS, understanding data collection, storage, and processing for large-scale analytics systems.
- Successfully practiced Amazon CloudWatch for monitoring system health via Metrics, Logs, and Alarms, improving observability and incident response.
- Understood Infrastructure as Code (IaC) benefits and how to use AWS CloudFormation to automate AWS infrastructure deployment with templates.
- Familiarized with AWS Cloud Development Kit (AWS CDK) for defining and deploying cloud infrastructure with programming languages flexibly and maintainably.
- Practiced Amazon DynamoDB, understanding NoSQL data models, table design, Partition Keys, Sort Keys, and basic CRUD operations.
- Learned using AWS Glue and Amazon Athena for data analysis and cost evaluation, supporting system performance optimization.
- Studied AWS IAM Federation with Microsoft Azure Active Directory for enterprise Single Sign-On (SSO).
- Understood differences among Savings Plans, Reserved Instances, and Reserved DB Instances for choosing appropriate cost optimization strategies.
- Understood Schema Conversion and Database Migration processes to AWS, laying groundwork for modernizing database systems in real projects.
- Reinforced Infrastructure as Code, Database, Data Analytics, Cost Optimization, and Cloud Migration knowledge for modern AWS infrastructure design, deployment, and management.

**Week 7:** 1 June – 5 June 2026

### Week 7 Objectives

- Learn AWS monitoring, administration, and infrastructure optimization tools.
- Practice advanced access management with IAM Permission Boundary.
- Learn AWS Systems Manager for managing multiple servers.
- Strengthen knowledge of cost optimization, data encryption, and building Data Lakes on AWS.

### Week 7 Outcomes

- Learned Amazon Managed Grafana for visual dashboards monitoring AWS resources via CloudWatch data.
- Understood IAM Permission Boundary for limiting maximum permissions granted to IAM Users or Roles.
- Successfully practiced AWS Systems Manager: Patch Manager, Run Command, and multi-instance administration automation.
- Understood EC2 Right Sizing principles for balancing performance and cost.
- Learned AWS KMS for creating and managing encryption keys and applying Encryption at Rest.
- Learned to track and visualize AWS costs for budget analysis and optimization.
- Understood basic Data Lake architecture on AWS for large-scale data storage, analytics, and AI foundations.
- Reinforced Monitoring, Security, System Administration, Cost Optimization, and Data Analytics for secure, efficient, scalable AWS infrastructure management.

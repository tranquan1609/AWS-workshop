**Week 4:** 11 May – 15 May 2026

### Week 4 Objectives

- Learn backup and data recovery solutions on AWS.
- Get familiar with migrating servers from on-premises to AWS.
- Practice deploying applications with Docker and AWS container services.
- Build automated deployment (CI/CD) pipelines for applications on Amazon ECS.

### Week 4 Outcomes

- Learned to use AWS Backup for backup and recovery plans for EBS, RDS, DynamoDB, and EFS, with automated backup for data safety.
- Understood VM Import/Export for migrating VMs between on-premises and AWS for infrastructure modernization.
- Successfully built Docker Images, deployed applications with Docker Containers, and understood container packaging and distribution benefits.
- Understood Amazon ECS architecture and deployment: creating Cluster, Task Definition, Service, and managing container application lifecycles.
- Learned CI/CD on AWS with automated pipelines to shorten release time, reduce manual work, and improve deployment stability.
- Reinforced Containerization, Orchestration, and DevOps knowledge, combining Docker, Amazon ECS, and CI/CD for modern software development and deployment on AWS.

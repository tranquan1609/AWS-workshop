**Week 3:** 4 May – 8 May 2026

### Week 3 Objectives

- Learn AWS management, monitoring, and support services.
- Get familiar with DNS on AWS through Amazon Route 53.
- Practice managing resources with AWS Command Line Interface (AWS CLI).
- Explore centralized identity management with AWS IAM Identity Center.

### Week 3 Outcomes

- Understood Amazon CloudWatch for monitoring AWS resources and applications, using Metrics, Logs, Alarms, and Dashboard.
- Learned AWS Support workflows, creating Support Cases, and understanding AWS Support Plans for system operations.
- Understood Amazon Route 53, DNS management on AWS, and Hybrid DNS connecting on-premises with Cloud.
- Practiced AWS CLI for resource management via command line, building a foundation for administration automation.
- Understood AWS IAM Identity Center for centralized identity management, configuring users, groups, and SSO access permissions.
- Reinforced knowledge of operations, monitoring, identity management, and system administration services on AWS for efficient, secure, and professional Cloud infrastructure management.

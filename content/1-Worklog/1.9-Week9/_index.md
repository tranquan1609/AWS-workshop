**Week 9:** 15 June – 19 June 2026

### Week 9 Objectives

- Strengthen knowledge of access management with AWS IAM.
- Learn Amazon Lightsail and cost-effective application deployment solutions.
- Practice deploying applications with Lightsail Container.
- Get familiar with AWS Step Functions workflow orchestration.
- Understand using IAM Roles to grant applications secure access to AWS services.

### Week 9 Outcomes

- Learned using IAM Roles with IAM Conditions for flexible access policies, controlling AWS resource access by specific conditions and strengthening security.
- Understood Amazon Lightsail characteristics for simple, cost-effective server deployment suitable for small-to-medium applications or test environments.
- Successfully practiced deploying applications with Amazon Lightsail Container, understanding container packaging and operations on Lightsail and differences from Amazon ECS.
- Understood AWS Step Functions for orchestrating automated workflows, building State Machines to coordinate multiple AWS services in serverless architecture.
- Learned using IAM Roles to grant applications secure AWS service access without storing Access Keys or Secret Keys in source code.
- Reinforced Identity & Access Management, Serverless Workflow, Container Services, and Cost Optimization for secure, automated, cost-effective AWS solutions.

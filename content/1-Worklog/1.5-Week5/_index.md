**Week 5:** 18 May – 22 May 2026

### Week 5 Objectives

- Learn advanced AWS security and networking services.
- Practice connecting multiple VPCs via VPC Peering and AWS Transit Gateway.
- Use AWS Lambda to automate Amazon EC2 cost optimization.
- Deploy CI/CD pipelines for applications on Amazon EC2 with AWS CodePipeline.

### Week 5 Outcomes

- Understood AWS Security Hub for centralizing security alerts, monitoring account security, and compliance assessment.
- Successfully practiced VPC Peering to connect two Amazon VPCs securely and efficiently.
- Learned AWS Transit Gateway architecture for centralized connectivity between multiple VPCs and on-premises networks.
- Used AWS Lambda with Amazon EventBridge to automate EC2 start/stop schedules for cost optimization.
- Practiced CI/CD with AWS CodePipeline for automated build and deployment to Amazon EC2.
- Reinforced Cloud Security, Networking, Serverless Computing, and DevOps knowledge for secure, scalable, and automated AWS systems.

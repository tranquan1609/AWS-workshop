**Week 10:** 22 June – 26 June 2026

### Week 10 Objectives

- Get familiar with cloud-based development using AWS Cloud9.
- Learn the process of converting Monolith applications to Microservices architecture.
- Practice building and deploying an independent Microservice.
- Understand data restructuring and workflows in Microservices architecture.
- Learn Amazon Kinesis for real-time data streaming and processing.

### Week 10 Outcomes

- Learned using AWS Cloud9 to develop, test, and manage source code directly on AWS, simplifying development environment setup.
- Understood Monolith-to-Microservices conversion, Monolith limitations, and benefits of splitting systems into independent services for scalability and maintainability.
- Practiced building a complete Microservice, understanding design, deployment, and management of independent services in distributed systems.
- Understood data restructuring and workflow methods in Microservices architecture, including database separation and inter-service processing flows to reduce coupling.
- Practiced Amazon Kinesis for collecting, streaming, and processing real-time data, understanding its role in large-scale analytics and event processing systems.
- Reinforced Cloud Development, Application Modernization, Microservices Architecture, and Real-time Data Processing for flexible, scalable, maintainable modern applications on AWS.

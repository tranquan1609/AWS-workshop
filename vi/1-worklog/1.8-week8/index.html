Mục tiêu tuần 8

  • Tiếp tục nghiên cứu kiến trúc Data Lake trên AWS và quy trình xử lý dữ liệu.
  • Tìm hiểu các dịch vụ Infrastructure as Code (IaC) trên AWS như CloudFormation và AWS CDK.
  • Thực hành xây dựng cơ sở dữ liệu NoSQL với Amazon DynamoDB.
  • Học cách phân tích chi phí, tối ưu hiệu năng và quản lý danh tính trong môi trường doanh nghiệp.
  • Tìm hiểu quy trình chuyển đổi và di dời cơ sở dữ liệu lên AWS.

Kết quả đạt được tuần 8

  • Tiếp tục củng cố kiến thức về Data Lake trên AWS, hiểu được quy trình thu thập, lưu trữ và xử lý dữ liệu phục vụ cho các hệ thống phân tích dữ liệu quy mô lớn.
  • Thực hành thành công với Amazon CloudWatch, biết cách theo dõi trạng thái hoạt động của hệ thống thông qua Metrics, Logs và Alarm, từ đó nâng cao khả năng giám sát và xử lý sự cố.
  • Hiểu được lợi ích của Infrastructure as Code (IaC) và biết cách sử dụng AWS CloudFormation để tự động hóa việc triển khai hạ tầng AWS thông qua các Template.
  • Làm quen với AWS Cloud Development Kit (AWS CDK), hiểu cách sử dụng ngôn ngữ lập trình để định nghĩa và triển khai hạ tầng Cloud một cách linh hoạt và dễ bảo trì.
  • Thực hành với Amazon DynamoDB, nắm được mô hình cơ sở dữ liệu NoSQL, biết cách thiết kế bảng dữ liệu, sử dụng Partition Key và Sort Key cũng như thực hiện các thao tác CRUD cơ bản.
  • Hiểu cách sử dụng AWS Glue và Amazon Athena để phân tích dữ liệu và đánh giá chi phí, đồng thời biết cách khai thác dữ liệu phục vụ việc tối ưu hiệu năng hệ thống.
  • Tìm hiểu cơ chế AWS IAM Federation kết hợp với Microsoft Azure Active Directory, hiểu cách triển khai xác thực tập trung (Single Sign-On) trong môi trường doanh nghiệp.
  • Nắm được sự khác biệt giữa Savings Plans, Reserved Instance và Reserved DB Instance, từ đó biết cách lựa chọn giải pháp tối ưu chi phí phù hợp với từng loại khối lượng công việc trên AWS.
  • Hiểu quy trình chuyển đổi lược đồ (Schema Conversion) và di dời cơ sở dữ liệu (Database Migration) lên AWS, tạo nền tảng cho việc hiện đại hóa hệ thống cơ sở dữ liệu trong các dự án thực tế.
  • Củng cố kiến thức về Infrastructure as Code, Database, Data Analytics, Cost Optimization và Cloud Migration, đồng thời nâng cao khả năng thiết kế, triển khai và quản lý hạ tầng AWS theo các tiêu chuẩn hiện đại.
